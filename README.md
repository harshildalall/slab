# SLAB.

Upload a photo of a road, bridge, or sidewalk. Get a full inspection report in seconds.

---

## What it does

- Identifies structural issues from a photo (cracks, spalling, corrosion, vegetation)
- Scores overall health 1-100
- Estimates time to urgent repair
- Generates prioritized maintenance recommendations
- Stores every inspection in HydraDB so you can track deterioration over time

## Stack

| Layer | Tool |
|---|---|
| Model | Kimi K2.5 via GMI Cloud |
| Orchestration | Dify |
| Memory | HydraDB |
| Frontend | HTML/CSS/JS |
| Hosting | GitHub Pages |
